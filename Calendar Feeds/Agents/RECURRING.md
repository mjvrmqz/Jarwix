# RECURRING — How to Handle Recurring Commitments

## Overview
Recurring commitments are stored in the **Recurring section of the Dashboard** in Notion. Claude checks for any recurring commitments at the start of every session before building a schedule. This file explains how to handle them.

## How to Handle Recurring Commitments
If there are recurring commitments in the Dashboard, treat them as fixed events — they cannot be moved or scheduled over. Reserve their full time blocks before placing any other tasks.

If there are no recurring commitments currently set, proceed normally. Do not assume the absence of data means the file is incomplete.

## What Counts as Recurring
Regular commitments on a weekly or monthly cadence — things like standing calls, family dinners, workout schedules, or any event that happens on a predictable schedule. If MJ mentions something recurring in conversation that isn't logged yet, ask if it should be added.
